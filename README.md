# Stack Overflow Survey Analysis

## 📊 Project Description
This project analyzes results from the Stack Overflow Developer Survey.

## 🔍 Key Analysis
- Total number of respondents
- Work experience statistics
- Remote work analysis
- Python popularity
- Salary analysis by country
- Age group analysis
- Industry trends among high-paid remote workers

## 🛠 Technologies
- Python
- pandas
- matplotlib

## 📁 Data
Stack Overflow Developer Survey dataset

## ▶️ How to Run
Open the notebook in Jupyter Notebook or Google Colab.
